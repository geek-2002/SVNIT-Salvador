## Technologies
* **HTML 5**

* **CSS 3**

* **Bootstrap 3 or 4**

* **Javascript**

* **MYSQL 2.18**

* **Node.js**

* **Express.js 4.17**

* **PUG**


## Setup

To run this project,install it locally using npm

```
$ npm install
$ npm start
```
